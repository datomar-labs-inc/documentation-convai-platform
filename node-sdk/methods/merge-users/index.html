Merge Users

mergeUsers() method is used to merge different super users into a single super user.

Important

By default, mergeUsers() will add the key value pairs of both the users in a single super user. If there is a conflict in the key for both the users, the user which was created first will hold priority. This can be overwritten in the request.

#Example Code

const {ConvaiAPIClient} = require('convai-sdk');


const convai = new ConvaiAPIClient(API_KEY);


convai.mergeUsers({
    superUserIds: [
        'ef44ad85-b97a-477b-b314-7702c32249be', '7c04daab-b694-48a1-a834-0f45af415b26'
    ],
        preferNewUserFields: ['firstName']
})
    .then(user => console.log(user))
    .catch(error => console.log(error));
