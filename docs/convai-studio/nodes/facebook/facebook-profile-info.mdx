## Usage

Facebook profile info node is used to get the user profile information from facebook. This only works if the user is either connected to the bot on the Facebook channel or has sent atleast 1 message to the bot in the past.

The node saves the profile information in the current execution scope. 

```json
{
    "fbProfile" : {
        "firstName": "firstNmae",
        "lastName" : "lastName",
        "gender": "gender",
        "locale": "locale",
        "name": "name",
        "profilePicURL": "url",
        "timezone": "0"
    }
}
```

So we can access the following information:

- First Name - ```{{data.fbProfile.firstName}}```
- Last Name - ```{{data.fbProfile.lastName}}```
- Gender - ```{{data.fbProfile.gender}}```
- Locale - ```{{data.fbProfile.locale}}```
- Name - ```{{data.fbProfile.name}}```
- Profile picture - ```{{data.fbProfile.profilePicURL}}```
- Timezone - ```{{data.fbProfile.timezone}}```
