## Usage
You can use the Slack Action Node to Get User Info or Add Reaction to the User Message.

## Get User Info
When Get User Info is executed. Slack returns a **[User Object](https://api.slack.com/types/user)** which is stored in the Execution Data **slackUserInfo**.

You access the Slack User Object by using ```{{data.slackUserInfo.[slack user object key]}}```

## Add Reaction
This Action adds a reaction to the message sent by user from Slack.

## Configuration
| Name          | Required |                      Description                                     |
| ------------- | -------- | -------------------------------------------------------------------- |
| Title         | Required | The title of the node                                                |
| Action Select | Required | Select Get User Info or Add Reaction |
| Reaction Select |  | When Add reaction is selected, this configuration allows the user to select reaction emoji of the reaction |
