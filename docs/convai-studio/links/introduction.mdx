Links determines how the flow goes to a node. For example, the flow goes to a node if the user input text is "yes". You can use links to control the flow via Data, Channel, User Input Text and etc.

There are two types of links on the Convai Platform - Omnichannel links and platform-specific links. Omnichannel links work on all the connected platforms and platform-specific links only work on the specified channel.
