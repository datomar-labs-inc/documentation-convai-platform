## Usage

This Link will be available when you have your **[Slack Integration](/convai-studio/integrations/channels/slack)** set.

The flow will execute via this link if the selected subtypes exists/doesn't exist. For more information, check **[Slack Message Events documentation](https://api.slack.com/events/message)**

## Configuration
| Name        | Required | Description                       |
|-------------|----------|-----------------------------------|
| Link Type   | Required | Type of the Link                  |
| Priority    | Required | Priority of the Link              |
| Check condition | Required | Check message subtype exists or not |
| Select Subtype | Required | Select and add Slack Message Subtype |
