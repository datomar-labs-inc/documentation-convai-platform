## Usage

Intent links works only if you have NLP Integration and **[NLP Node](/convai-studio/nodes/nlp/nlp)** set. Currently Convai supports **[DialogFlow](/convai-studio/integrations/nlp/dialogflow)** and **[LUIS]()** integrations.

Intent link determines the flow once the selected intent is matched. It is usally connected from an Input Node or Any Time System node to another node. You can use the Intent Selector in the Link's configuration to select an Intent.

## Configuration

| Name            | Required | Description               |
|-----------------|----------|---------------------------|
| Link Type       | Required | Type of the Link          |
| Priority        | Required | Priority of the Link      |
| Check Operation | Required | Check Operation of Intent |
| Intent Name     | Required | Name of Intent            |
| Intent Selector | Optional | Selector of Intent        |
