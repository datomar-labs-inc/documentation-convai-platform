## Usage
This Link will be available when you have your **[Twilio Integration](/convai-studio/integrations/channels/twilio)** set.

The flow will execute via this link if there is no speech in the twilio voice call.

## Configuration
| Name        | Required | Description                       |
|-------------|----------|-----------------------------------|
| Link Type   | Required | Type of the Link                  |
| Priority    | Required | Priority of the Link              |
