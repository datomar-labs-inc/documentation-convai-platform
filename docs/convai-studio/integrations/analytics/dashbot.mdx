This document outlines the process to integrate Dashbot with Convai Studio.

1. Go to Settings and click edit on an environment you would like to configure.

:::tip Note
_**NOTE: If you are doing a new integration, add it into the Development environment before any other environments.**_
:::

2. In the integration settings, click on Add Integration and select Dashbot. You will see something like this:
<img alt="Add Dashbot Integration" src={useBaseUrl('img/convai-studio/integrations/analytics/dashbot/image1.png')} />

3. Click on Dashbot and a dialog like below will open
<img alt="Dashbot Dialog" src={useBaseUrl('img/convai-studio/integrations/analytics/dashbot/image5.png')} />

4. Go to your Dashbot account and click on Add Chatbot. Add: A bot name, Platform as **Universal**, choose a category and production status.
<img alt="Add Dashbot Integration" src={useBaseUrl('img/convai-studio/integrations/analytics/dashbot/image4.png')} />

5. After you click on continue, copy the API key
<img alt="Add Dashbot Integration" src={useBaseUrl('img/convai-studio/integrations/analytics/dashbot/image3.png')} />

6. Go back to Convai and paste the API key.

7. Dashbot is now connected to the environment. For usage, see dashbot nodes.
