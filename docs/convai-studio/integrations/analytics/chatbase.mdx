This document outlines the process to integrate Chatbase with Convai Studio.

1. Go to Settings and click edit on an environment you would like to configure.

:::tip Note
_**NOTE: If you are doing a new integration, add it into the Development environment before any other environments.**_
:::

2. In the integration settings, click on Add Integration and select Chatbase. You will see something like this:
<img alt="Add Chatbase Integration" src={useBaseUrl('img/convai-studio/integrations/analytics/chatbase/image5.png')} />

3. Click on Chatbase and a dialog like below will open
<img alt="Chatbase Dialog" src={useBaseUrl('img/convai-studio/integrations/analytics/chatbase/image3.png')} />

4. Go to your Chatbase account and click on Add a Bot. Add: A bot name, Country, Industry and Business Case.
<img alt="Add Chatbase Integration" src={useBaseUrl('img/convai-studio/integrations/analytics/chatbase/image4.png')} />

5. After you click on continue, copy the API key
<img alt="Add Chatbase Integration" src={useBaseUrl('img/convai-studio/integrations/analytics/chatbase/image1.png')} />

6. Go back to Convai and paste the API key.

7. Chatbase is now connected to the environment. For usage, see chatbase nodes.
