Flaggable class is used for reference. It stores the data of the user once it is fetched from the API.

## Public Methods

#### ```public get<T>(key: string): T```

The get() method returns the data for the key

| Name | Type   | Attribute | Description     |
| ---- | ------ | --------- | --------------- |
| key  | string |           | Name of the key |

#### ```public clear()```

The clear() method clears all the data
