The ```DeleteSuperUser()``` method is used to delete a channel of a super user. It returns a response and an error (error will not be nil if there's an error). The response is a **[SuperUser]()** type that is the super user deleted.

The ```DeleteSuperUser()``` method expects the ID of super user which is a UUID type. You can get the super user ID in ```Users > Click on the user > find the super user id above User Data section on the right```. Parse the string value to UUID by using ```uuid.FromStringOrNil```.


#### Example Code
```go
package main

import (
    "fmt"
    "encoding/json"
    "github.com/datomar-labs-inc/convai-sdk-go"
    uuid "github.com/satori/go.uuid"
)

const API_KEY = "YOUR API KEY"

func main() {
    convaiClient := convai.NewAPIClient(API_KEY)
	// parse the string super user ID to UUID
	superUserId := uuid.FromStringOrNil("88628ae5-b7b1-4d49-b65c-26a2a6c6b4d3")
	res, err := convaiClient.DeleteSuperUser(superUserId)

	if err != nil {
		fmt.Println(err.Error())
	} else {
		prettyRes, _ := json.MarshalIndent(&res, "", "\t")
		fmt.Println(string(prettyRes))
	}
}
```
