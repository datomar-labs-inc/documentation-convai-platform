The ```UpdateUserData()``` method is used to update the user data of the super user. It returns a response and an error (error will not be nil if there's an error). The response is a **[SuperUser]()** type that is the super user has User Data updated.

The ```DeleteSuperUser()``` method expects the ID of super user which is a UUID type and the input which is **[```UpdateUserDataInput```]()** type. You can get the super user ID in ```Users > Click on the user > find the super user id above User Data section on the right```.

:::important
Even though the API sends back the updated user instantly, their is a delay of **1 sec** to update the user data in the database.
:::

#### Example Code
```go
package main

import (
    "fmt"
    "encoding/json"
    "github.com/datomar-labs-inc/convai-sdk-go"
)

const API_KEY = "YOUR API KEY"

func main() {
    convaiClient := convai.NewAPIClient(API_KEY)

	// setup the data to be deleted and updated
	var deleteData []string
	deleteData = append(deleteData, "lastName")
	var updateData map[string]interface{}
    updateData["foo"] = "bar"
    
    //update user data
	res, err := convaiClient.UpdateUserData("d56e7906-136c-4944-a8cc-73645d8389a1",&convai.UpdateUserDataInput{
		Set:    updateData,
		Delete: deleteData,
	})

	if err != nil {
		fmt.Println(err.Error())
	} else {
		prettyRes, _ := json.MarshalIndent(&res, "", "\t")
		fmt.Println(string(prettyRes))
	}
}
```
