Flaggable class is used for reference. It stores the data of the user once it is fetched from the API.

Refer to **[Flaggable](../types#flaggable)** for more type information.

#### ```(f *Flaggable) Get(key string) (value interface{}, present bool)```
The Get() method returns the value for a key. If the value does not exist, it returns (nil, false)

| Name   | Type   |Description               |
| ------ | ------ |------------------------- |
| key | string | key of the variable |

#### ```(f *Flaggable) Clear()```
The Clear() method clears all the data.
