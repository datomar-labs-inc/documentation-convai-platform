The Request Context stores the context of a single convai request.

Refer to **[RequestContext](../types#requestcontext)** for more type information.

#### ```(r *RequestContext) Error(err ExecError)```
The Error() method appends an **[ExecError](../types#execerror)** to the RequestContext, and set it as the last error.

| Name   | Type   |Description               |
| ------ | ------ |------------------------- |
| r | \*RequestContext | RequestContext type |
