Merge Users

Use MergeUsers() method to merge different super users into a single super user. It returns a response and an error (error will not be nil if there's an error). The response is a SuperUser type.

Important

By default, MergeUsers() will add the key value pairs of both the users in a single super user. If there is a conflict in the key for both the users, the user which was created first will hold priority. This can be overwritten in the request.

#Example Code

In practical situations, you will not need to know the ID of every Super User in order to merge. Use QueryUsers() method to get a set of users.

package main


import (
    "fmt"
    "encoding/json"
    "github.com/datomar-labs-inc/convai-sdk-go"
    uuid "github.com/satori/go.uuid"
)


const API_KEY = "YOUR API KEY"


func main() {
    convaiClient := convai.NewAPIClient(API_KEY)
    
    // query users that has User Scope data "firstName" exists
    userQuery := convai.UserQueryBuilder().All().Where("firstName").Exists().Build()
    users, err := convaiClient.QueryUsers(userQuery)


    if err != nil {
        fmt.Println(err.Error())
    } else {
        // extract queried user IDs
        var userIDs []uuid.UUID
        for _, user := range users.Users {
            userIDs = append(userIDs, user.ID)
        }
        
        // merge users
        res, err := convaiClient.MergeUsers(&convai.MergeUsersRequest{
            SuperUserIDs: userIDs,
            PreferNewUserFields: []string{"firstName"},
        })


        if err != nil {
            fmt.Println(err.Error())
        } else {
            prettyRes, _ := json.MarshalIndent(&res, "", "\t")
            fmt.Println(string(prettyRes))
        }
    }
}
