Delete Channel User

The DeleteChannelUser() method is used to delete a channel of a super user. It returns a response and an error (error will not be nil if there's an error). The response is a ChannelUser type that is the channel user deleted.

The DeleteChannelUser() method expects the ID of channel user which is a String type. You can get the channel user ID in Users > Click on the user > find the super user id above Seesion Data section on the right

#Example Code

package main


import (
    "fmt"
    "encoding/json"
    "github.com/datomar-labs-inc/convai-sdk-go"
)


const API_KEY = "YOUR API KEY"


func main() {
    convaiClient := convai.NewAPIClient(API_KEY)
    res, err := convaiClient.DeleteChannelUser("dev-console-81085beb-a6fd-48dd-ad09-522da3b6b2511548")


    if err != nil {
        fmt.Println(err.Error())
    } else {
        prettyRes, _ := json.MarshalIndent(&res, "", "\t")
        fmt.Println(string(prettyRes))
    }
}
