Add Channel User Node

#Usage

The Add Channel User node is used to add another channel for a super user. Usually we use this node to add Twilio SMS or Twilio Voice channel to the super user.

For example, let's assume a user is talking to the bot on Dev Console and it provides its phone number. Now in order to reach it via sms in future, you can add a channel for it and save it.

It is important to note that Add Channel User automatically adds the new channel in the same Super User so we don't need to use the Merge Users node.

#Bonus

A good practice of this node is add Twilio Voice Channel to the super user when the user is talking to the bot through Twilio SMS channel, and vice versa.

#Configuration

The configuration of the node is as defined below:

NameRequiredDescription
TitleRequiredThe title of the node
Twilio SMSOptionalThe Twilio SMS channel to be added
Twilio VoiceOptionalThe Twilio Voice channel to be added
Force Add Channel UserOptionalForce to add a channel user even if that type of channel is added to the super user
