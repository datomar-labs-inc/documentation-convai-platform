Slack Action Node

#Usage

You can use the Slack Action Node to Get User Info or Add Reaction to the User Message.

#Get User Info

When Get User Info is executed. Slack returns a User Object which is stored in the Execution Data slackUserInfo.

You access the Slack User Object by using {{data.slackUserInfo.[slack user object key]}}

#Add Reaction

This Action adds a reaction to the message sent by user from Slack.

#Configuration

NameRequiredDescription
TitleRequiredThe title of the node
Action SelectRequiredSelect Get User Info or Add Reaction
Reaction SelectWhen Add reaction is selected, this configuration allows the user to select reaction emoji of the reaction
