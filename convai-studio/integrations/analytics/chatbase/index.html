Chatbase Integration

This document outlines the process to integrate Chatbase with Convai Studio.

  1. Go to Settings and click edit on an environment you would like to configure.
Note

NOTE: If you are doing a new integration, add it into the Development environment before any other environments.

  1. In the integration settings, click on Add Integration and select Chatbase. You will see something like this:

  2. Click on Chatbase and a dialog like below will open

  3. Go to your Chatbase account and click on Add a Bot. Add: A bot name, Country, Industry and Business Case.

  4. After you click on continue, copy the API key

  5. Go back to Convai and paste the API key.

  6. Chatbase is now connected to the environment. For usage, see chatbase nodes.
